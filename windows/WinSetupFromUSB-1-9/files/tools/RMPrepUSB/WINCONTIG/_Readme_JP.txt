WinContig は、あなたの PC にインストールする必要なく利用できる スタンドアロンのデフラグメンテーションツールです、いかなるレジストリエントリも作成しません。

このツールの主要な目的は、ディスク全体ではなくファイル単位で迅速にデフラグを行うことです。

WinContig では、プロファイルをファイルとして保存、分類することが可能で プログラムの動作を制御する為の多くのコマンドラインスイッチ(任意)が用意されています。

WinContig は、Windows XP, Windows Vista Windows 7 で使用する為に設計されたアプリケーションです。

実行するためには Windows XP または Windows Vista Windows 7 ユーザは、管理者権限でログオンする必要があります。

==================================================================== 
インストール
====================================================================
WinContig はスタンドアロンで実行可能なプログラムです。インストール処理や追加のDLLを必要としません。使用を開始するには、ダウンロードしたZIPファイルを、任意の場所で解凍して中に含まれている実行ファイルを実行するだけです。

注意: WinContig の古いバージョンを使用していて、新しいバージョンにアップデートする場合は、先に古いバージョンを手動で削除する必要があります。 
また、新しいバージョンを最初に実行する時に、すべての設定はデフォルト値で開始されます。
