WinContig è uno strumento che consente di deframmentare i file desiderati senza la necessità di dovere deframmentare il disco intero. Per deframmentare i file, è sufficiente aggiungere in elenco i file e/o le cartelle che si desiderano deframmentare e quindi fare clic sul pulsante Deframmenta. È anche possibile analizzare lo stato di frammentazione dei file. WinContig consente la creazione di profili, utili ad esempio se si deframmentano spesso gli stessi gruppi di file e, inoltre, accetta una serie di parametri facoltativi della riga di comando che consentono di controllare il funzionamento del programma. Il programma è stand-alone, non necessita di alcuna libreria .dll esterna particolare e non scrive alcuna informazione nel Registro di sistema.

WinContig funziona sui sistemi operativi Windows XP, Windows Vista, Windows Seven e Windows 8/8.1.

Come utente di Windows XP, Windows Vista, Windows Seven o Windows 8/8.1, è necessario disporre dei privilegi amministrativi completi.

==================================================================== 
Installazione
====================================================================
WinContig è un'applicazione stand-alone che non richiede installazione o librerie DLL addizionali. Per utilizzare WinContig, estrarre i file contenuti nell'archivio compresso "WContig.zip" in una cartella di propria scelta e quindi eseguire il file "WinContig.exe".

NOTA: prima di installare la versione corrente di WinContig, rimuovere la versione precedente del programma. La prima volta che si esegue WinContig, le opzioni del programma saranno impostate ai valori predefiniti.
