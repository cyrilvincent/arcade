Visão geral do utilitário
O Utilitário de controle do número de série do processador Intel® é um programa do Windows® que ativa ou desativa a leitura do número de série do processador Pentium (R) III feita pelo software. Isso permite que os usuários controlem quais programas de software ou sites da Web terão permissão para ler o número de série do processador. Após instalado, o utilitário é executado automaticamente cada vez que o sistema é iniciado.

Este utilitário coloca um ícone na bandeja do sistema Windows® para fornecer ao usuário uma indicação visual do estado do recurso de número de série do processador. O usuário tem a opção de ocultar o ícone da bandeja do sistema. A desativação do número de série do processador pode ser feita a qualquer momento, mas a ativação do número de série requer que o sistema seja reiniciado.

Requisitos do sistema
O Utilitário de controle de número de série do processador Intel requer
o Um sistema com processador Pentium III 
o Windows 95, Windows 98 ou Windows NT 4.0 (ou posterior?)
o 2 megabytes em disco rígido

Instalação
Execute o setup.exe no diretório em que você descompactou o arquivo. Observação: A tentativa de instalação deste utilitário em um sistema que não contenha um processador Pentium III gerará uma mensagem de erro.

Visão geral do número de série do processador
O número de série do processador Intel é um novo recurso do processador Pentium III que funciona como um identificador para o processador. O número de série do processador foi projetado para ser exclusivo e, quando utilizado em conjunto com outros métodos de identificação, pode ser usado para identificar o sistema ou o usuário. Este número pode ser utilizado com vários aplicativos  que se beneficiam de formas mais potentes de indentificação do sistema e de usuários. 

O número de série do processador é análogo a um número de série convencional, porém apresenta várias diferenças importantes.
o Pode ser lido por aplicativos de software 
o A leitura pode ser desativada através de programas utilitários, como este, ou através do BIOS, dependendo da configuração do sistema.

Para obter informações adicionais sobre o processador Pentium III e sobre o número de série do processador, visite www.intel.com/pentiumiii

Perguntas freqüentes sobre o número de série do processador e utilitário de controle.

1. Quais são os benefícios do número de série do processador?
Resposta: O número de série do processador será utilizado em aplicativos que se beneficiam de formas mais potentes de indentificação do sistema e de usuários, como:
o Aplicativos que utilizam recursos de segurança
o Gerenciamento de ativos
o Bate-papo seguro

2. Por que eu iria querer desativar o número de série do meu processador?
Resposta: A Intel acredita que o número de série do processador irá oferecer benefícios reais aos usuários e está desenvolvendo recursos que, em conjunto com o número de série do processador, permitem que os provedores de serviços responsáveis ofereçam serviços que mantenham a sua privacidade. 
No entanto, se estiver preocupado com o modo pelo qual um determinado aplicativo/serviço, que esteja utilizando o número do processador, possa comprometer a sua privacidade, você tem a opção de desativá-lo através do Utilitário de controle de número de série do processador Intel.

3. Qual o estado padrão do número de série do processador?
Resposta: O estado predefinido do número de série do processador è ativado, atè quando o controle do número de série do processador for instalado. Uma vez instalado, o Utilitário de controle de número de série do processador ira DESATIVÁ-LO por definição. O usuário poderá ATIVAR o número de série do processador uitilizando o Utilitário. 

4. Um site da Web pode ler o meu número de série sem que eu saiba?
Resposta: Não, normalmente isto não acontece. Os sites da Web não podem ler o número de série sem a sua permissão para fazer o download de um programa que possa ler o número de série do processador. Quase todos os navegadores estão configurados para avisar os usuários sempre que eles fizerem o download de um software executável. A menos que tenha desativado o aviso no seu navegador, você deve receber uma notificação.

5. A Intel rastreia os números de série?
Resposta: Normalmente não, somente quando relacionados ao processo de fabricação. A Intel não irá, a menos que tenha permissão expressa e prévia de um usuário, coletar dados do número de série, relacionados a um usuário.


6. Que programas e/ou sites da Web utilizam atualmente o número de série do processador?
Resposta: Uma lista completa dos programas que se beneficiam do número de série do processador e de outros novos recursos do processador Pentium III pode ser encontrada em 
http://www.intel.com/portugues/pentiumiii/utility.htm 

7. Como é possível saber se o número de série do processador está ativado?
Resposta: A grande maioria dos sistemas com processadores Pentium III é fornecida com o número de série do processador ativado. O utilitário de controle permite que você verifique o estado: 
 
1) através do próprio ícone (o ícone desativado mostra um círculo vermelho com um "x" na cor branca),
2) selecionando o item de menu "Estado" ao clicar no ícone da bandeja de tarefas, ou 
3) a partir da dica de ferramenta mostrada quando o mouse é posicionado sobre o ícone da bandeja de tarefas.
