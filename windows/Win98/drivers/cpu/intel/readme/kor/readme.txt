유틸리티 개요 
Intel(R) 프로세서 일련 번호 제어 유틸리티는 Pentium (R) III의 일련 번호 읽기 기능을 
소프트웨어를 사용하여 사용 가능 또는 사용 중지하도록 제어하는 Windows(R)용 
프로그램입니다. 소프트웨어 프로그램이나 웹 사이트에 따라서 이 프로세서 일련 번호의 
읽기 권한을 부여할 수 있습니다. 이 유틸리티를 설치하면 시스템을 시작할 때마다 
자동으로 실행됩니다.

이 유틸리티의 아이콘이 Window(R) 시스템 트레이에 작성되므로 사용자들이 
프로세서 일련 번호 기능의 상태를 쉽게 확인할 수 있습니다. 이 시스템 
트레이 아이콘을 숨길 수도 있습니다. 프로세서 일련 번호 기능을 언제든지 
사용 중지할 수 있으며 이 기능을 다시 사용하려면 시스템을 다시 시작해야 
합니다.

시스템 요구 사양
프로세서 일련 번호 제어 유틸리티를 사용하려면 아래와 같은 사양이 필요합니다.
- Pentium III 프로세서가 장착된 시스템
- Windows 95, Windows 98, Windows NT 4.0 (또는 이상 버전)
- 하드 디스크 2 메가바이트

설치
파일의 압축을 푼 디렉토리에서 setup.exe를 실행하십시오. 주의: Pentium III 프로세서가 
없는 시스템에서 이 유틸리티를 설치하려고 하면 오류 메시지가 발생합니다.

프로세서 일련 번호 개요
Intel 프로세서 일련 번호는 Pentium III 프로세서의 새 기능으로 프로세서를 
확인해 줍니다. 프로세서 일련 번호는 사용자마다 고유한 번호로 다른 확인 방식과
함께 사용하여 시스템이나 사용자를 확인할 수 있습니다. 이 번호는 시스템 및 사용자 확인이
필요한 다양한 응용 프로그램에서 사용할 때 보다 안전하고 강력한 기능을 제공합니다.

프로세서 일련 번호는 기존의 일련 번호와 유사하지만 다음과 같은 점에서 
차이가 납니다.
- 소프트웨어 응용 프로그램으로 읽을 수 있습니다.
- 유틸리티 프로그램이나 또는 시스템 구성에 따라서 BIOS를 통해 읽기 기능을 
사용 중지할 수 있습니다.

Pentium III 프로세서 및 프로세서 일련 번호에 대한 추가 정보는 www.intel.com/pentiumiii 
사이트를 참고하십시오.

프로세서 일련 번호 및 제어 유틸리티에 대한 질문과 대답(FAQ)

1. 프로세서 일련 번호의 장점은 무엇입니까?
프로세서 일련 번호는 다음과 같이 시스템 및 사용자 확인이 필요한 다양한 
응용 프로그램에서 사용할 때 보다 안전하고 강력한 기능을 제공합니다.
- 보안 기능을 사용하는 응용 프로그램
- 자산 관리 프로그램
- 보안이 필요한 채팅 프로그램

2.  어떤 경우에 프로세서 일련 번호 기능을 사용 중지해야 합니까?
프로세서 일련 번호는 여러 가지면에서 사용자들에게 유용한 기능이므로 사용을 권장합니다. 
Intel에서는 서비스 공급업체들이 보안 기능이 포함된 서비스를 안심하고 제공할 수 있도록 
프로세서 일련 번호와 관련된 기능을 개발하고 있습니다. 그러나 지정된 응용 프로그램이나 
서비스에서 일련 번호를 사용하는 방식이 사용자의 보안에 영향을 미칠 수 있다고 
우려되는 경우에는 Intel 프로세서 일련 번호 제어 기능을 사용 중지하실 수 있습니다.

3. 프로세서 일련 번호의 초기 상태는 무엇입니까?
프로세서 일련 번호 제어 유틸리티를 설치하기 전에 프로세서 일련 번호는 사용 가능한 상태로 기본 설정되어 있습니다. 프로세서 일련 번호 제어 유틸리티를 설치하고 나면 이 기능이 자동으로 사용 중지됩니다. 나중에 이 유틸리티를 사용하여 프로세서 일련 번호를 사용 가능하게 할 수 있습니다.

4. 사용자가 모르게 웹 사이트에서 사용자의 일련 번호를 읽을 가능성은 없습니까?
불가능합니다. 사용자가 프로세서 일련 번호를 읽을 수 있는 프로그램을 다운로드할 수 
있도록 허용하지 않는 한 웹 사이트에서 사용자의 일련 번호를 읽을 수 없습니다. 
대부분의 브라우저는 웹 사이트에서 실행 소프트웨어를 다운로드하려고 할 때마다 
사용자에게 경고 메시지를 보내도록 구성되어 있습니다. 사용자가 이 경고 기능을 
사용 중지한 경우를 제외하고는 이 메시지를 받아 볼 수 있습니다. 

5. Intel에서 일련 번호를 추적합니까?
제조 과정에 관련된 경우를 제외하고는 추적하지 않습니다. Intel에서는 사용자가 사전에 
동의하지 않는 한 일련 번호 데이터를 수집하지 않습니다.

6. 프로세서 일련 번호를 현재 사용하는 프로그램이나 웹 사이트는 무엇입니까?
프로세서 일련 번호를 사용하는 프로그램의 목록 및 Pentium III 프로세서의 
기타 새로운 기능이 www.intel.com/kr/hangul/pentiumiii/utility.htm 에 
소개되어 있습니다.

7. 프로세서 일련 번호 기능이 사용 가능한지 어떻게 알 수 있습니까?
Pentium III 프로세서가 장착된 대부분의 시스템에서는 프로세서 일련 번호가 
사용 가능합니다. 제어 유틸리티에서 다음과 같이 확인할 수 있습니다.
 
1) 아이콘 확인(사용 중지된 아이콘은 붉은 원안에 흰색 "x"가 표시되어 나타남).
2) 작업 트레이 아이콘을 클릭하여 "상태" 메뉴 항목 선택
3) 마우스를 상태 트레이 아이콘 위에 올려 놓을 때 표시되는 도구 설명 참고
