                                 HOW TO PLAY
                                 -----------
* Press Time button to view current time (hold to view alarm time).
* Press Game A button to play game A (easy).
* Press Game B button to play game B (hard).
* Instructions:
  - Olive throws food items for Popeye, and has to catch them.
  - Try to catch the food items. Move Popeye left or right by pressing
    the corresponding control buttons.
  - When a food item falls into the ocean he it is a half miss.
  - Popeye also has to avoid attacks from Bluto.
  - When Bluto hits Popeye, Popeye will fall into the ocean
    and this is a miss.
  - Three misses and the game ends.
* Score:
  - 1 point is scored for each food item caught by Popeye.
* Maximum displayed score is 999.
* When the score reaches a bonus score (200 or 500 points), any miss marks
  indicated at that time are erased with a fanfare and game the continues.
* Highest scores are saved in the System Registry.
* 30 sec after game ends, time mode is displayed over a demo animation
  of the game.



                                   CONTROLS
                                   --------
A) by MOUSE and TOUCH SCREEN

   Right-click anywhere to activate the popup menu

   Game A 		Game A (hold to view hiscore)
   Game B		Game B (hold to view hiscore)
   Time 		Time (hold to view alarm time)
   ACL			Reset the game

   Left/Right buttons	Move Popeye left/right

B) by KEYBOARD

   1 (*)		Game A (hold to view hiscore)
   2 (*)		Game B (hold to view hiscore)
   3 (*)		Time (hold to view alarm time)
   5 (*)		Reset the game
   SPACEBAR (*)		Pause/Resume the game (in-game only)

   Arrow keys (*)	Move Popeye left/right

   F1			View "About"
   F2			View "Readme.txt"
   F3			View "Whatsnew.txt"
   F4			View instructions manual browser
   F12			Change video mode (fullscreen/windowed)

   Esc			Quit program

(*) you can redefine the keys from the panel featured in the popup menu
