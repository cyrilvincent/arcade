                                 HOW TO PLAY
                                 -----------
* Press Time button to view current time (hold to view seconds).
* Press Game A button to play game A (easy).
* Press Game B button to play game B (hard).
* Instructions:
  - A large scale earthquake occurred!
  - Rescue refugees on the top of the buidings by helicopter.
  - If the helicopter collides against red building or tower on either side,
    the game is over.
  - Also if you fail to rescue 2 refugees the game is over.
  - In game A, helicopter can carry one only refugee at one time.
  - In game B, helicopter can carry up to two refugees at the same time.
* Score:
  - 10 points per each refugee taken to the safety building (on the right side
    of the screen).
* Maximum display score is 9990 points. At 9990 points the game is over
  automatically.
* Highest scores are saved in the System Registry.



                                   CONTROLS
                                   --------
A) by MOUSE and TOUCH SCREEN

   Right-click anywhere to activate the popup menu

   Game A 		Game A (hold to view hiscore)
   Game B		Game B (hold to view hiscore)
   Time 		Time (hold to view seconds)
   ACL			Reset the game

   Left/Right buttons	Fly the helicopter left/right

B) by KEYBOARD

   1 (*)		Game A (hold to view hiscore)
   2 (*)		Game B (hold to view hiscore)
   3 (*)		Time (hold to view seconds)
   5 (*)		Reset the game

   Arrow keys (*) 	Fly the helicopter left/right

   F1			View "About the author..."
   F2			View "Readme.txt"
   F3			View "Whatsnew.txt"
   F4			View instructions manual browser
   F12			Change video mode (fullscreen/windowed)

   Esc			Quit program

(*) you can redefine the keys from the panel featured in the popup menu
