                                 HOW TO PLAY
                                 -----------
* Click the outer panel to open the handheld game.
* Hold "Time" button to view alarm time (set to 12:00 PM)
  Press and release "Time" button to view current time
* Hold "Game A" or "Game B" to view highscores for games A or B
  Press and release "Game A" (easy) or "Game B" (hard) buttons to play
* Instructions:
  - Mario is back in action with his brother Luigi, working in a bottling
    plant, loading cases on a belt conveyor.
  - The conveyor carries the cases up for loading onto trucks.
  - Operate Mario and Luigi up and down ladders in order to get cases.
  - When a case is dropped, the foreman get angry and a miss is scored.
  - Rhree misses and game is over
* Score:
  -  1 point per case loaded on a conveyor.
  - 10 points per truck loaded with 8 cases.
* At 300 points fanfare sounds and all misses are canceled; if there are no
  misses when 300 points are reached, game goes into "CHANCE TIME". Score
  flashes and all points are awarded at double value until a miss is made.
* Maximum displayed score is 999 points.
* Highest scores are saved in the System Registry.
* After game ends, time function displays after about 5 minutes.



                                   CONTROLS
                                   --------
A) by MOUSE and TOUCH SCREEN

   Right-click anywhere to activate the popup menu

   Game A 		Game A (hold to view hiscore)
   Game B		Game B (hold to view hiscore)
   Time 		Time (hold to view alarm time)
   ACL			Reset the game

   Button overlays (*)	Move Mario and Luigi

B) by KEYBOARD

   1 (**)		Game A (hold to view hiscore)
   2 (**)		Game B (hold to view hiscore)
   3 (**)		Time (hold to view alarm time)
   5 (**)		Open the game/Reset the game

   Arrow keys		Move Mario and Luigi up/down

   Q (**)		Move Luigi up
   A (**)		Move Luigi down
   P (**)		Move Mario up
   L (**)		Move Mario down

   F1			View "About the author..."
   F2			View "Readme.txt"
   F3			View "Whatsnew.txt"
   F4			View instructions manual browser
   F12			Change video mode (fullscreen/windowed)

   Esc			Quit program

(*)  the controller overlay can be enabled/disabled at will (default: enabled)
(**) you can redefine the keys from the panel featured in the popup menu
