TXT2K7 !
--------

C'est un kit de deux petit programmes qui permettent de récupérer des
 programmes en format text issus de capture en OCR ou directement tapés
 avec le Bloc-Note.


Pour charger le programme compilé, tappez:
LOAD""
ou
LOAD"CASS:"
Si vous avez séléctionné le lecteur de disquette.

Nota:
* Les sources en visual basic sont incluses dans le zip.

Syntaxe:
--------


TXT2K7_TO7 [nom_du_fichier_texte.TXT]
TXT2K7_MO5 [nom_du_fichier_texte.TXT]

Dans les deux cas, placer le nom entier avec l'extension.
