WinContig ist ein einfach zu bedienendes, eigenständiges Defragmentierungwerkzeug, das keine Installationsverzeichnisse oder Registry-Einträge auf dem Computer erstellt. Damit können einzelne Dateien schnell defragmentiert werden, ohne den gesamten Datenträger defragmentieren zu müssen. Darüber hinaus ermöglicht WinContig, Dateien in Profile zu gruppieren und akzeptiert eine Reihe von optionalen Befehlszeilenoptionen für die Programmsteuerung.

WinContig läuft auf Windows XP, Windows Vista, Windows 7 und Windows 8/8.1 Betriebssysteme.

Benutzer von Windows XP, Windows Vista, Windows 7 oder Windows 8/8.1 müssen über volle Administratorrechte verfügen.

Installation
====================================================================
WinContig ist eine eigenständige ausführbare Datei, die keine Installation oder zusätzliche DLLs erfordert. Um sie zu nutzen, extrahieren Sie den Inhalt des ZIP-Archivs in einen beliebigen Ordner und doppelklicken Sie die ausführbare Datei.

HINWEIS: Vor der Installation der aktuellen Version von WinContig muss die alte Version entfernt werden. Bei der ersten Ausführung werden alle Einstellungen auf die Standardwerte zurückgesetzt.
