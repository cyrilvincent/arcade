Compatibilité de fichier avec les versions précédentes de Microsoft Excel 
-------------------------------------------------------------------------
Microsoft Excel version 4.0 lit tous les documents Microsoft Excel version 2.x et 3.0 existants. Cependant, les documents enregistrés sous le format Microsoft Excel version 4.0 ne peuvent pas être ouverts dans Microsoft Excel version 3.0a ou antérieure. Pour ouvrir un document dans une version précédente de Microsoft Excel, choisissez la commande Fichier Enregistrer sous et sélectionnez le format de fichier « Excel 3.0 » ou « Excel 2.1 ». NOTE: Vous risquez de perdre une partie de la mise en forme quand le document est enregistré sous le format d'une version précédente.


-----------------------------------------------------------------------
Compatibilité de macro avec les versions précédentes de Microsoft Excel
-----------------------------------------------------------------------
Les macros développées dans les versions précédentes de Microsoft Excel sont totalement compatibles avec Microsoft Excel version 4.0.


-----------------------------------------------------------------
Message de dépassement mémoire après avoir quitté Microsoft Excel
-----------------------------------------------------------------
Si vous utilisez Microsoft Windows version 3.0, et que vous recevez un message indiquant un dépassement mémoire lorsque vous tentez de lancer une application après avoir quitté Microsoft Excel, essayez de relancer l'application. Cela ne se produit pas sous Microsoft Windows version 3.1.


----------------------------------------------------------------
Modification de la commande Path dans votre fichier AUTOEXEC.BAT 
----------------------------------------------------------------
Si vous ne voulez pas que Microsoft Excel Install modifie votre fichier AUTOEXEC.BAT, vous pouvez le modifier vous-même. En particulier, vous devez modifier la commande Path pour y inclure le répertoire contenant le programme Microsoft Excel (EXCEL.EXE) et ses fichiers associés. 

Pour modifier la commande Path, ouvrez le fichier AUTOEXEC.BAT dans un éditeur de texte ou un traitement de texte. Pour ajouter un répertoire dans la liste, ajoutez le chemin d'accès complet (lettre de l'unité de disque et nom du répertoire). Les chemins d'accès doivent être séparés par des points-virgules. Pour ajouter un répertoire appelé EXCEL, ajoutez ;c\excel à la ligne de AUTOEXEC.BAT qui commence par « path ». Par exemple,

   path c:\dos;c:\windows;c:\excel

Note: Si vous exécutez Microsoft Excel à partir du serveur d'un réseau,
----- vous devez inclure le répertoire du serveur qui contient EXCEL.EXE
      dans la commande Path du fichier AUTOEXEC.BAT de votre station de
      travail.

Lorsque vous avez fini de modifier le fichier, enregistrez-le sous le format texte seul et redémarrez votre ordinateur pour que les modifications soient prises en compte.

Pour plus d'informations sur la commande Path et le fichier AUTOEXEC.BAT, consultez votre documentation MS-DOS.


----------------------------------------------------------------------------
Exécution des didacticiels à partir d'un répertoire du type lecture-écriture
----------------------------------------------------------------------------
Pour exécuter les didacticiels, Microsoft Excel doit enregistrer les paramètres de l'environnement en cours. Pour cela, le répertoire en cours doit être du type lecture-écriture. S'il ne l'est pas, un message d'erreur vous indiquera que Microsoft Excel ne peut enregistrer les informations et les didacticiels ne s'exécuteront pas.


-------------------------------------------------
Fonctions DIALOGUE.OUVRIR et DIALOGUE.ENREGISTRER
-------------------------------------------------
Ces deux fonctions macro ont été omises par inadvertance dans le manuel de référence des fonctions de Microsoft Excel. Ces fonctions deviennent disponibles lorsque vous installez la macro complémentaire FICHFNCT.XLA. Vous pouvez en apprendre plus sur les fonctions dans l'Aide. La manière la plus rapide d'accéder à ces informations est de :

  1. Cliquer deux fois sur l'outil Aide.

     Microsoft Excel affiche la boîte de dialogue Rechercher de l'Aide.

  2. Taper dans la zone de texte : fonctions de fichier

  3. Choisir « Atteindre ».


------------------------------------
Compatibilité avec Q+E et SQL Server 
------------------------------------
Si vous utilisez SQL Server version 4.2 ou une version ultérieure et si vous avez installé Q+E, supprimez les deux fichiers suivants de votre répertoire Excel :

   DBNMP3.DLL
   W3DBLIB.DLL

Ces versions des fichiers ne sont pas compatibles avec SQL Server version 4.2. De nouvelles versions de ces fichiers sont installées avec SQL Server version 4.2 et doivent être copiées dans votre répertoire \WINDOWS\SYSTEM où ils peuvent être utilisés par Q+E ainsi que par toute application SQL Server.


------------------------------------------------------------
Word pour Windows version 2.0 et Microsoft Excel version 4.0 
------------------------------------------------------------
Si vous avez l'intention d'utiliser la commande Impression Fusion de Microsoft Word pour Windows version 2.0 avec une feuille de calcul de Microsoft Excel version 4.0, vous devez d'abord enregistrer la feuille de calcul sous le format de fichier de Excel 3.0.
