实用程序概述

Intel(R) 处理器序号控制实用程序用于 Windows(R) 程序，让您可通过此软件启用或禁用读取 Pentium (R) III 处理器的序号。这将允许用户控制何种软件程序或 Web 站点已经许可读取的处理器序号。安装完成后，实用程序在系统每次启动时将自动运行。

此实用程序在 Windows(R) 系统任务栏中置入一个图标，可提供给用户一个处理器序号功能状态的视觉指示。用户也可以选择隐藏此系统任务栏图标。用户可以在任何时候禁用处理器序号功能，但是若要重新启用处理器序号功能，则必须重新启动系统。

系统要求 
Intel(R) 处理器序号控制实用程序要求
- 以 Pentium III 处理器为基础的系统
- Windows 95、Windows 98、Windows NT 4.0 （或更新的系统？）
- 2 MB 的硬盘空间

安装
从您解压缩文件的目录中运行 setup.exe。注意：如果试图在一个没有 Pentium III 处理器的系统上安装此实用程序，系统将产生错误讯息。

处理器序号概述
Intel 处理器序号是 Pentium III 处理器的新功能，它可以作为处理器的标识符。根据设计，每一处理器都有单一的序号，且在与其他的标识方法同时使用时，可用于标识系统或用户。此序号能用于广泛的各种不同类型的应用程序，有助于加强它们系统和用户标识的方式。

处理器序号与传统的序号相比有类似之处，但是有几点重要的区别。
- 它可被软件应用程序读取。
- 依据系统的配置，可通过此实用程序的程序或通过 BIOS 来禁用读取。

欲知更多的关于 Pentium III 处理器和处理器序号的详细信息，请访问：www.intel.com/pentiumiii。

处理器序号常见问题。

1. 处理器序号的优点是什么？
1答：处理器序号将用于各种应用程序中，有助于加强系统和用户标识的方式，例如
- 使用安全性能的应用程序
- 资产管理
- 安全性对话

2.  为什么我要关闭我的处理器序号？
2答：Intel 相信，处理器序号将提供给用户极大的好处，可以开发与处理器序号共同使用的功能，让有责任的服务提供商提供保护您隐私的服务。然而，如果您认为使用处理器序号的特定应用程序/服务可能干涉您的隐私，您可以选择使用 Intel 处理器序号控制实用程序关闭处理器序号。

3. 处理器序号的默认状态是什么？
3答：在处理器序号控制实用程序安装完成前，根据默认设置，处理器序号处于开启状态。一旦处理器序号控制实用程序安装完成后，根据默认设置，实用程序将关闭处理器序号。用户可使用实用程序打开处理器序号。
 
4. Web 站点有可能在我不知道的情况下读取我的序号吗？
4答：不能，一般来说不能。除非您允许 Web 站点下载可以读取处理器序号的程序到您的系统中，Web 站点将不能读取您的序号。几乎所有的浏览器在用户下载可执行软件时，都会警告用户。除非您在浏览器中禁用警告功能，否则您应该收到浏览器的通知。

5. Intel 会追踪序号吗？
5答：除了牵涉到制造过程，一般来说不追踪序号。Intel 将不会在用户事先不同意的情况下，收集可用于用户标识的序号数据。

6. 哪些程序和/或 Web 站点当前可使用处理器序号？
6答：一份完整的利用 Pentium III 处理器的处理器序号和其他新功能的程序清单可以在下列网址中找到： 
www.intel.com/cn/gb/pentiumiii/utility.htm 

7. 我如何知道我已打开我的处理器序号？
7答：绝大多数以 Pentium III 处理器为基础的系统在送交给用户时，都已启用处理器序号。控制实用程序让您以下列方式检查您的处理器状态：
 
1) 图标本身（禁用图标会显示出一个有白色“X”的红色圆圈），
2) 单击任务栏图标，选择“状态”菜单项，或
3) 将鼠标移到任务栏图标时所显示的工具提示。
