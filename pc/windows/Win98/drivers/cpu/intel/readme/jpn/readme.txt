プロセッサ シリアル ナンバ コントロール ユーティリティの概要
Intel(R) プロセッサ シリアル ナンバ コントロール ユーティリティは、
ソフトウェアによる Pentium(R) III プロセッサのシリアル ナンバ
読み取り機能を有効または無効に切り替える Windows(R) プログラムです。
ユーザーはこの機能を使用して、ソフトウェアまたは Web サイトに対する
プロセッサのシリアル ナンバの読み取り許可を制御することができます。
このユーティリティをインストールすると、システムを起動するたびに
ユーティリティが自動的に実行されます。

Windows(R) のシステム トレイにユーティリティのアイコンが表示されるので、
プロセッサ シリアル ナンバ機能のステータスが一目でわかります。
このシステム トレイ アイコンは非表示にすることもできます。
プロセッサ シリアル ナンバはいつでも無効にできますが、有効にするには
システムを再起動する必要があります。

システム必要条件
Intel プロセッサ シリアル ナンバ コントロール ユーティリティの
システム必要条件は次の通りです。
- Pentium III プロセッサが搭載されたシステム
- Windows 95、Windows 98、または Windows NT 4.0 (またはそれ以降)
- 2 MB のハードディスク容量

インストール
ファイルを解凍したディレクトリにある setup.exe を実行します。
注意 : Pentium III プロセッサが搭載されていないシステムにこのユーティリティを
インストールしようとすると、エラー メッセージが表示されます。

プロセッサ シリアル ナンバの概要
Intel プロセッサ シリアル ナンバはプロセッサの識別子として使用される、
Pentium III プロセッサの新機能です。
プロセッサ シリアル ナンバはそれぞれ固有にデザインされています。
ほかの識別方法と合わせて使用することで、システムまたはユーザーを識別することが
できます。プロセッサ シリアル ナンバは、システムおよびユーザーを識別する際に
詳細な情報を必要とするさまざまなアプリケーションで使用可能です。

プロセッサ シリアル ナンバは規定のシリアル ナンバに類似しますが、
いくつかの重要な相違点があります。
- ソフトウェア アプリケーションを使用して読み取りが可能です。
- システムの構成によって、ユーティリティ プログラムを使用または
BIOS を通じて読み取り機能を無効にすることが可能です。


Pentium III プロセッサおよびプロセッサ シリアル ナンバの詳細については、
www.intel.com/pentiumiii をご覧ください。

プロセッサ シリアル ナンバおよびコントロール ユーティリティに関する質問 (FAQ)

1. プロセッサ シリアル ナンバの利点は？
1A. プロセッサ シリアル ナンバは、システムおよびユーザーを識別する際に詳細な情報を
必要とする、次のようなアプリケーションで使用できます。
- セキュリティ機能を使用したアプリケーション
- 資産管理
- 安全なチャット

2. プロセッサ シリアル ナンバを無効にする理由は？
2A. インテルでは、プロセッサ シリアル ナンバを登録していただくと、
ユーザーの皆様にさまざまな特典を提供させていただきます。さらに、
プライバシーの保護を厳守したサービスを提供するサービス プロバイダが
使用できるプロセッサ シリアル ナンバの新機能を開発中です。
提供されるアプリケーションおよびサービスによってプロセッサ シリアル ナンバが
使用されるとプライバシーの保護に影響を与えると心配される方は、
Intel プロセッサ シリアル ナンバ コントロール ユーティリティを使用して
シリアル ナンバを無効にできます。

3. プロセッサ シリアル ナンバのデフォルト ステータスは？
3A. プロセッサ シリアル ナンバのステータスは、プロセッサ シリアル ナンバ コントロール ユーティリティがインストールされるまで、デフォルトによりオンに設定されています。プロセッサ シリアル ナンバ コントロール ユーティリティがインストールされると、プロセッサ シリアル ナンバはデフォルトによりオフに設定されます。ユーザーは、このユーティリティを使用してプロセッサ シリアル ナンバをオンに切り替えることができます。

4. Web サイトで、知らないうちにシリアル ナンバが読み取られることはありますか?
4A. 通常は読み取られることはありません。
Web サイト側に、プロセッサ シリアル ナンバを読み取り可能なプログラムの
ダウンロードを許可した場合を除いて、Web サイトからはシリアル ナンバを
読み取ることはできません。ほとんどの Web ブラウザは、実行可能ファイルを
ダウンロードする際、ユーザーに警告メッセージを表示するように設定されています。
ただし、この設定が無効になっている場合は、警告メッセージが表示されないので
注意してください。

5. インテルでは、シリアル ナンバのトラッキング (追跡) を行っていますか?
5A. 製品処理に関する場合を除いて、通常はトラッキングを行っていません。
インテルでは、ユーザーの同意なしにシリアル ナンバに関するデータを収集することは
ありません。

6. 現在プロセッサ シリアル ナンバを使用しているプログラムおよび Web サイトは
ありますか?
6A. プロセッサ シリアル ナンバを使用して特典が取得できるプログラムのリスト、
および Pentium III プロセッサに関するその他の新機能については、
www.intel.co.jp/jp/pentiumiii/utility.htm をご覧ください。

7. プロセッサ シリアル ナンバが有効になっていることを確認するには?
7A. Pentium III プロセッサが搭載されているほとんどのシステムでは、
プロセッサ シリアル ナンバは有効に設定されています。
プロセッサ シリアル ナンバ コントロール ユーティリティを使用して
次のいずれかの方法でステータスが確認できます。
 
1) アイコンを確認します。無効の場合、アイコンに赤いマルと白い X が付きます。
2) タスク トレイのアイコンをクリックして [ステータス] メニューを表示して確認します。
3) タスク トレイのアイコン上にマウスのポインタを置くとツール ヒントに
ステータスが表示されます。
