公用程式簡介
Intel(R) 處理器序號控制公用程式為用於 Windows(R) 作業系統的程式，
讓您可藉由此軟體來啟用或停用讀取 Pentium(R) III 處理器的序號。經由
此功能，使用者可控制何種軟體程式或網站可有權限讀取系統的處理器序號。
在安裝後，此公用程式會在每次系統啟動時自動執行其作業。

此公用程式會在 Windows(R) 系統工作列內置入圖示，以提供使用者辨識
處理器序號功能的狀態。使用者亦可選擇隱藏系統工作列圖示。使用者可在
任何時候停用處理器序號功能，但若要重新啟用此功能，則需重新啟動系統。

系統需求
Intel 處理器序號控制公用程式需有
- 以 Pentium III 處理器為基礎的系統
- Windows 95、Windows 98、Windows NT 4.0 (或以上?)
- 2 MB 硬碟空間

安裝
請自您解壓縮檔案的目錄內執行 setup.exe。注意：若嘗試將此公用程式
安裝於沒有 Pentium III 處理器的系統上，系統將會產生錯誤訊息。

處理器序號簡介
Intel 處理器序號為 Pentium III 處理器的新功能，可用來做為一種處理器
的驗證方式。根據設計，每一處理器皆有其獨一的序號，且可在與其他驗證方式
一同使用時，用來辨識系統或使用者。此一序號可用於許多能利用較嚴格之
系統與使用者驗證方式的應用程式內。

處理器序號與傳統的序號相似，但有以下數點重要不同處。
- 可供軟體應用程式讀取序號。
- 視系統配置，可經由類似此公用程式的程式或經由 BIOS 來停用序號讀取。

處理器序號與控制公用程式常見問題

1. 處理器序號的優點是什麼？
1答：您可將處理器序號用於可使用較嚴格之系統與使用者驗證方式的
應用程式內，如
- 可使用安全驗證功能的應用程式
- 資產管理
- 安全性對話

2. 我為什麼會想要關閉我的處理器序號？
2答：Intel 相信處理器序號可提供使用者極大的好處，並正開發可與處理器
序號一同使用的功能，以讓服務提供者可在提供服務時保有您的隱私。但是，
若您考慮到使用處理器序號的應用程式/服務有可能侵犯到您的隱私權，您可
使用 Intel 處理器序號控制公用程式來關閉此項功能。

3. 處理器序號的預設狀態是什麼？
3答：依據預設，在您安裝處理器序號控制公用程式前，處理器序號為處於啟用狀態。在您安裝處理器序號控制公用程式後，此程式會依預設停用處理器序號功能。使用者可用此公用程式來啟用處理器序號。

4. 網站有可能在我不知情的情況下讀取我的處理器序號嗎？
4答：一般說來不會。除非您事先允許網站下載可讀取處理器序號的程式
到您的系統內，否則網站不可能讀取您的序號。幾乎所有的瀏覽器都會在下載
可執行軟體前先行警告使用者。除非您停用了您瀏覽器的警告功能，否則
您應會在下載軟體前先收到瀏覽器的通知。

5. Intel 會追蹤序號嗎？
5答：除了有關生產處理過程外，一般說來不會。Intel 不會在沒有事前經過
使用者同意的情況下，逕自收集可用來辨識使用者的序號資料。


6. 目前有什麼程式與/或網站可使用處理器序號？
6答：您可自以下網站，找到一份目前可利用處理器序號的程式清單，以及
Pentium III 處理器的新功能：
www.intel.com/tw/big5/pentiumiii/utility.htm 

7. 我如何知道我有啟用我的處理器序號？
7答：絕大多數 Pentium III 處理器的系統在送交給顧客時皆已啟用處理器
序號。控制公用程式可讓您經由以下方式檢查您的處理器狀態：

1) 圖示本身 (停用時圖示會顯示出一個有白色「x」符號的紅色圓圈)，
2) 按一下工作列圖示後選取 [狀態] 功能表，或
3) 在將游標移至工作列圖示上時所顯示的工具提示。
