Übersicht über das Dienstprogramm
Das Intel(R) Dienstprogramm zur Steuerung der Prozessor-Seriennummer ist ein Windows*-Programm, mit dem das Lesen 
der Seriennummer des Pentium(R) III-Prozessors durch die Software aktiviert oder deaktiviert wird. Auf diese Weise können Anwender 
steuern, welche Softwareprogramme oder Websites die Berechtigung zum Lesen
der Prozessor-Seriennummer erhalten. Nach der Installation wird die Utility automatisch mit jedem Systemstart ausgeführt.

Dieses Dienstprogramm plaziert ein Symbol in der Windows*-Systemablage, um den Anwender auf visuelle Weise über den Status der Prozessor-Seriennummer-Funktion in Kenntnis zu setzen. Das Systemablagensymbol kann optional ausgeblendet werden. Die Prozessor-Seriennummer läßt sich jederzeit deaktivieren. Zum Aktivieren dieser Funktion muß jedoch das System neu gestartet werden. 
Systemanforderungen
Das Intel(R) Dienstprogramm zur Steuerung der Prozessor-Seriennummer erfordert folgende Hard- und Software:
- Ein System auf Basis des Pentium(R) III-Prozessors
- Windows 95, Windows 98, Windows NT 4.0 (oder höher?)
- 2 MB Festplattenspeicher

Installation
Führen Sie die setup.exe-Datei in dem Verzeichnis, in dem Sie die Datei dekomprimiert haben. Hinweis:  Wenn Sie versuchen, dieses Dienstprogramm 
auf einem System zu installieren, das nicht mit einem Pentium(R) III-Prozessor ausgestattet ist, erhalten Sie eine Fehlermeldung.

Prozessor-Seriennummer - Übersicht
Bei der Intel Prozessor-Seriennummer handelt es sich um eine neue Funktion des Pentium(R) III-Prozessors, die zur 
Identifizierung des Prozessors eingesetzt wird. Die Prozessor-Seriennummer ist eindeutig und kann in Verbindung mit anderen Identifikationsmethoden zum Identifizieren des Systems oder Anwenders verwendet werden. Diese Nummer läßt sich in einer Vielzahl von Anwendungen einsetzen, die dadurch 
von der erhöhten Sicherheit bei der System- und Anwenderkennung profitieren.

Die Prozessor-Seriennummer entspricht einer konventionellen Seriennummer, jedoch mit folgenden wichtigen Unterschieden: 
- Sie kann durch eine Softwareanwendung gelesen werden. 
- Das Lesen der Nummer kann über Dienstprogramme wie dieses oder durch das BIOS, je nach Systemkonfiguration, deaktiviert werden. 

Weitere Informationen über den Pentium(R) III-Prozessor und die Prozessor-
Seriennummer erhalten Sie unter www.intel.com/pentiumiii.

Prozessor-Seriennummer - FAQ

1.  Was sind die Vorteile des Prozessor-Seriennummer?
1A. Die Prozessor-Seriennummer wird in Anwendungen eingesetzt, die 
dadurch von der erhöhten Sicherheit bei der System- und Anwenderkennung profitieren, zum Beispiel 
- Anwendungen mit Sicherheitsfunktionen 
- Bestandsverwaltung 
- Sichere Chat-Anwendungen

2.  Wann wäre das Deaktivieren der Prozessor-Seriennummer angebracht?
2A. Intel glaubt, daß Anwendern mit der Prozessor-Seriennummer zwingende Vorteile geboten werden. Von Intel werden Funktionen in Verbindung mit der Prozessor-Seriennummer entwickelt, die es verantwortungsbewußten Service-Providern ermöglichen, Dienste zum Bewahren Ihrer Privatsphäre bereitzustellen. 
Sollten Sie jedoch Bedenken haben, daß die Verwendung der Prozessor-Seriennummer durch eine bestimmte Anwendung oder einen Service Ihre Privatsphäre beeinträchtigen könnte, haben Sie die Möglichkeit, diese Funktion mit Hilfe des 
Intel(R) Dienstprogramms zur Steuerung der Prozessor-Seriennummer zu deaktivieren.

3. Was ist der standardmäßige Status der Prozessor-Seriennummer?
3A. Standardmäßig ist die Prozessor-Seriennummer solange aktiviert, bis das Dienstprogramm zur Steuerung der Prozessor-Seriennummer installiert wird. Sobald dieses Dienstprogramm installiert ist, wird die Prozessor-Seriennummer standardmäßig deaktiviert. Der Anwender kann die Prozessor-Seriennummer dann mit Hilfe des Dienstprogramms wieder aktivieren. 

4. Kann eine Website meine Prozessor-Seriennummer ohne mein Wissen lesen? 
4A. Nein, in der Regel nicht. Websites können Ihre Prozessor-Seriennummer nur dann lesen, wenn Sie das Downloaden eines Programms gestatten, mit dem die Prozessor-Seriennummer gelesen werden kann. Nahezu alle Browser sind so konfiguriert, daß Anwender vor dem Herunterladen von ausführbarer Software gewarnt werden. Wenn Sie die Warnfunktion Ihres Browser nicht deaktivieren, sollten Sie immer eine Benachrichtigung erhalten. 

5. Führt Intel Protokoll über Prozessor-Seriennummern?
5A. Im Allgemeinen nicht, ausgenommen, es handelt sich um den Herstellungsprozeß. Intel erfaßt ohne eine vorherige und ausdrückliche Einwilligung des Anwenders keine Seriennummerndaten, die gleichzeitig mit einem Anwender identifiziert werden können. 

6. Welche Programme und/oder Websites verwenden zur Zeit die Prozessor-Seriennummer?
6A. Eine vollständige Liste von Programmen, die die Prozessor-Seriennummer und 
andere neue Funktionen des Pentium(R) III-Prozessors nutzen, finden Sie unter 
http://www.intel.com/deutsch/pentiumiii/utility.htm.

7. Wie stelle ich fest, ob meine Prozessor-Seriennummer aktiviert ist?
7A. Die überwiegende Mehrzahl von Systemen auf Basis des Pentium(R) III-Prozessors wird mit aktivierter 
Prozessor-Seriennummer geliefert. Mit Hilfe des Steuerprogrammes können Sie den Status wie folgt überprüfen: 
 
1) Die Anzeige des Symbols (ein deaktiviertes Symbol wird mit einem roten Kreis und einem weißen "x" angezeigt)
2) Durch Auswahl des Menübefehls "Status", indem Sie auf das Systemablagensymbol klicken. 
3) Durch die Anzeige des QuickInfos, wenn Sie die Maus auf dem Systemablagensymbol positionieren.
