Focal V3d for the KIM-1

This is the programming language Focal v3.d program for the KIM-1 computer.

Documentation can be found at:

http://www.hansotten.com - 6502 Info pages - KIM-1

The binary files in this archive have been reproduced from the original 
cassette recording with the tool KIMTAPE, http://www.alphalink.com.au/~edsa/ 
This program also makes it possible to reproduce original cassette recordings
that can be read by a KIM-1.

Available files in this archive

focalzp.bin 	ID=01 Start $0000 length 256 bytes zeropage of the program
focalm.bin 	ID-02 start $2000 length 5809 bytes main program
