Installation :

Il suffit simplement de désarchiver le fichier compressé sur un lecteur.

Le programme l'auto-configurera selon le répertoire d'installation, vérifiera la présence des fichiers utile à sont exécution et si besoin est, demandera l'accord afin de créer des fichiers utiles au bon déroulement du programme.

(A noter qu'aucune information n'est enregistrer dans la base de registre, cela facilitera donc la désinstallation.)

  • 'DskCenter.exe' – Programme compilé en exécutable.

  • 'Disk.def' – Liste des disquettes disponibles, utilisable avec CPCemu pour créer une disquette vierge.

  • 'Mess Disk format.txt' – Base de donnée Mess pour l'affichage des compatibilités système.

  • 'Info DskCenter.txt' – Fichier d'information en Anglais.

  • 'zip32.dll' – Fichier de libraire dynamique pour la compression des fichiers ( lire Info DskCenter.txt pour plus d'informations)

  • 'unzip32.dll' - Fichier de libraire dynamique pour la décompression des fichiers ( lire Info DskCenter.txt pour plus d'informations)

  • Divers répertoire pour l'organisation des fichiers esclaves :

    Softwares > Répertoire par défaut des différents systèmes.

    Icones > Répertoire des icônes par défaut des différents systèmes.

    CPCformat, CPNformat > Répertoire de sortie des fichiers convertis aux formats respectif.

    Exported files > Répertoire de sortie des Fichiers CPC en mode texte.

    Zip Tags > Répertoire d'archivage des commentaires Zip.

    Boot > Répertoire d'archivage pour les fichiers contenant les différents systèmes reconnus (saut PC DOS, bootstraps variables).

    Language > Répertoire linguistique composé de fichiers ressource du type Fr.lng, De.lng ...

    Nota: Pour changer le fichier ressource il suffit de changer le fichier de configuration comme suit : Fichier DskCenter.ini

    [setup]

    Language=FR

    (Pour les nostalgiques de l'Anglais, il suffit de ne rien mettre après le égale)

    Temp > Répertoire temporaire pour la décompression des fichiers sélectionnés dans l'explorateur DskCenter.



Mise à jour :

Il est important de garder les fichiers de configuration déjà mis à jour :

soit,

  • pour les émulateurs : Emulator_list.ini (liste des arguments pour le lancement des applications)

  • Pour DskCenter : DskCenter.ini (liste des Fichiers préférences, dossier de travail ...)

  • Pour les commentaires de fichiers compressés : \Zip Tags\ZipTag.ini (liste des fichiers texte)

    Dans ce cas, il est recommander d'ouvrir « WinZip » et de décompresser à la demande ou d'archiver les fichiers « ini » dans le menu du programme et de les rappeler ultérieurement.

Désinstallation

Il suffit d'effacer le répertoire concerné. (Sans oublier de déplacer les fichiers contenus dans le répertoire « Softwares », s'il existe des images disques à l'intérieur.)
