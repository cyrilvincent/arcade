  Goblins   for Atari ST

This is Gamex version for easier
game save/load when playing from
hard disk. 2MB RAM req.

When want to continue from saved state
be sure that start from same DIR as when
saved state !

By pressing M at HW detect screen may
enter machine setting if have Mega STE
or Falcon.
Only if you know what are doing !

Game will work  with 1MB machines,
but without gamex.
